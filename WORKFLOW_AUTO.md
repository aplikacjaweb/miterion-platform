# WORKFLOW_AUTO.md

This file is for automated workflows and protocols. It ensures that required startup files are read and maintained after memory compaction.

## Protocols
- Ensure all required startup files are read at the beginning of each session.
- Create missing files if they do not exist to maintain continuity.

## Required Files
- `WORKFLOW_AUTO.md`
- `memory/YYYY-MM-DD.md` (daily memory logs)

## Actions
- If a required file is missing, create it immediately.
- Log all significant actions and decisions in the daily memory file.
